Сделай красивый современный сайт для изучения латинских слов. Данные берём из src/data/flashcards_ru_la_by_lesson.json. Использовать только section === "Лексика". Всего 13 минимумов (занятия 1..13).

Стек (обязательно)

React + TypeScript + Vite

TailwindCSS

React Router

Данных/серверов нет — всё статично.

Состояние/прогресс можно хранить в localStorage.

1) ДАННЫЕ И НУМЕРАЦИЯ
1.1 Загрузка

Прочитай flashcards_ru_la_by_lesson.json и собери:

decks[1..13] — массив карточек для каждого минимума

allCards — общий массив всех карточек (конкатенация минимумов 1..13 по порядку)

1.2 Глобальный индекс карточки (важно!)

Каждая карточка должна иметь:

globalIndex (начиная с 1) — сквозная нумерация по allCards

lesson — номер минимума (1..13)

ru, la

Пример: allCards[0].globalIndex = 1, allCards[1399].globalIndex = 1400.

2) РОУТЫ

/ — главная (выбор разделов + общий режим)

/all — режим “Все карточки”

/lesson/:id — режим “Карточки минимума N”

/quiz/:id — блиц-викторина для минимума N (30 вопросов)

3) ГЛАВНАЯ СТРАНИЦА /
UI

Заголовок

Большая кнопка “Все карточки” → /all

Сетка из 13 плиток: “Лексический минимум 1…13”

показать количество слов

две кнопки на каждой плитке:

“Карточки” → /lesson/N

“Блиц (30)” → /quiz/N

4) РЕЖИМ КАРТОЧЕК (ОБЩИЙ /all И МИНИМУМ /lesson/:id)

Это один и тот же компонент, только источник карточек разный.

4.1 Внешний вид и поведение

На экране одновременно одна карточка по центру.

Лицевая сторона: русский

При перевороте: латинский

Переворот по клику на карточку + кнопка “Перевернуть”.

Анимация flip 3D rotateY (плавная).

4.2 Навигация — строго так:

Слева большая кнопка-стрелка “◀ Предыдущая”

Справа большая кнопка-стрелка “Следующая ▶”

Обязательно поддержать клавиши клавиатуры:

← предыдущая

→ следующая

Space — переворот

НИКАКИХ “листаний снизу” (никаких длинных списков, скролла карточек и т.п.)

4.3 Поле “Перейти к карточке”

Самое важное:

Над карточкой (или под заголовком) сделать строку:

Перейти к № [input] [Перейти]

Пользователь вводит глобальный номер (например 1400).

После Enter или кнопки “Перейти”:

В режиме /all: показать карточку с globalIndex = введённый

В режиме /lesson/:id: если введённый индекс не относится к этому минимуму, показать сообщение:

“Карточка №1400 не входит в минимум N. Введите номер из диапазона X–Y.”

где X–Y — глобальный диапазон для этого минимума.

После прыжка навигация стрелками листает дальше по порядку: 1401, 1402, ...

Также отображать текст:

Карточка №1400 из 2150 (для /all)

или Карточка 12 из 40 (глобально №1400) (для минимума)

4.4 Защита от ошибок

Если введено меньше 1 или больше длины массива — показать ошибку.

Ввод должен принимать только числа.

4.5 Дополнительно (желательно)

Кнопка “Перемешать порядок” (но по умолчанию порядок прямой)

Показать “Минимум N” и “Глобальный №” в маленьком бейдже.

5) БЛИЦ-ВИКТОРИНА /quiz/:id
5.1 Суть

Для выбранного минимума N сделать квиз из ровно 30 вопросов (если слов меньше 30 — тогда вопросов столько же, сколько слов).

Вопросы выбираются рандомно из слов этого минимума без повторов.

5.2 Формат вопроса

На экране показывать русское слово (вопрос).

Ниже 4 кнопки-ответа (латинский вариант):

1 правильный

3 неправильных — тоже рандомные из этого же минимума (без повторов, не совпадают с правильным)

5.3 Поведение при ответе (ЖЁСТКО)

После выбора варианта:

Правильный вариант всегда становится зелёным

Все неправильные варианты становятся красными

Дальше выбор блокируется (нельзя менять ответ)

Появляется кнопка “Следующий вопрос”

Показать счёт: Правильно: X / Y

5.4 Перемешивание

Вверху кнопка “Перемешать блиц”

полностью заново генерирует 30 вопросов и варианты

сбрасывает счёт и прогресс

5.5 Завершение

Когда вопросы закончились:

Экран результатов:

“Вы закончили: X / 30”

кнопки:

“Ещё раз (перемешать)” → генерирует новый блиц

“Назад к минимуму” → /lesson/N

“На главную” → /
